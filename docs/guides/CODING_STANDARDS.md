# コーディング規約

## コミットメッセージ（gitmessage）

以下の絵文字を先頭に付けてコミットメッセージを作成します。

### Emojis

- `🐛 :bug:` バグ修正
- `🎈 :balloon:` 文字列変更や軽微な修正
- `👍 :+1:` 機能改善
- `✨ :sparkles:` 部分的な機能追加
- `🎉 :tada:` 盛大に祝うべき大きな機能追加
- `♻️ :recycle:` リファクタリング
- `🚿 :shower:` 不要な機能・使われなくなった機能の削除
- `💚 :green_heart:` テストやCIの修正・改善
- `👕 :shirt:` Lintエラーの修正やコードスタイルの修正
- `🚀 :rocket:` パフォーマンス改善
- `🆙 :up:` 依存パッケージなどのアップデート
- `🔒 :lock:` 新機能の公開範囲の制限
- `👮 :cop:` セキュリティ関連の改善
- `🔧 :wrench:` 設定関連変更
- `📝 :memo:` ドキュメントの整理
- `🚧 :construction:` 作業中

### Format

`絵文字 scope: 説明`

- 説明は日本語で簡潔に記述します。
- scope は変更対象を短く示します（例: `docs` `ai` `claude` `root`）。

必要に応じて本文を続けます。
