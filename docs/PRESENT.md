# 抽選イベント ルールまとめ

## 基本ルール
- 応募は **1000pt = 1口**
- **1回の抽選で応募できるのは最大3口まで**
- チャネルポイント特典は「1口」特典（1000pt）のみ
  - 複数口応募したい場合は最大3回まで実行可能

---

## サブスク特典（口数ボーナス）

### ボーナス計算式
**ボーナス口数 = 累計サブスク月数 × Tier係数 × 1.1 ÷ 3（切り上げ）**

- 小数点は **切り上げ**
- **最低ボーナス**: サブスク登録者は計算結果が0でも最低1口のボーナス

### Tier係数
| Tier | 係数 |
|------|------|
| Tier1 | 1.0 |
| Tier2 | 1.1 |
| Tier3 | 1.2 |
| サブスク未登録 | ボーナスなし（0口） |  

---

## 最終口数の計算式
**最終口数 = 応募口数（最大3） + ボーナス口数**

- ボーナス口数は上記の計算式で算出
- 最終口数の小数点は **切り上げ**

---

## 計算例

### サブスク未登録の人
- 応募口数：3口（リワード3回実行）
- ボーナス：0口
- **最終：3口**

### 抽選月だけTier1にした人（累計1ヶ月）
- 応募口数：3口（リワード3回実行）
- ボーナス計算：1ヶ月 × 1.0 × 1.1 ÷ 3 = 0.366... → 切り上げで1口
- ボーナス：1口（最低ボーナス適用）
- **最終：3 + 1 = 4口**

### 半年継続しているTier1（累計6ヶ月）
- 応募口数：3口（リワード3回実行）
- ボーナス計算：6ヶ月 × 1.0 × 1.1 ÷ 3 = 2.2 → 切り上げで3口
- ボーナス：3口
- **最終：3 + 3 = 6口**

### 半年継続しているTier2（累計6ヶ月）
- 応募口数：3口（リワード3回実行）
- ボーナス計算：6ヶ月 × 1.1 × 1.1 ÷ 3 = 2.42 → 切り上げで3口
- ボーナス：3口
- **最終：3 + 3 = 6口**

### 1年継続しているTier3（累計12ヶ月）
- 応募口数：3口（リワード3回実行）
- ボーナス計算：12ヶ月 × 1.2 × 1.1 ÷ 3 = 5.28 → 切り上げで6口
- ボーナス：6口
- **最終：3 + 6 = 9口**

---

## 注意事項
- 抽選は配信内で行います
- サブスク累計月数とTierは Twitch の情報を元に確認します
- チャンネルポイントは返却不可のため、応募前に口数をよく確認してください
