# ドキュメント

このディレクトリは、確定済みの仕様・運用ガイドを保持するだす。
揮発的な計画・手順・進捗は GitHub Issues で管理するだす。

## 構成

- `docs/guides/GETTING_STARTED.md`: AI運用の初期セットアップ手順
- `docs/guides/AI_INSTRUCTION_PORTING.md`: AI運用ルール移植ガイド
- `docs/guides/ISSUE_WORKFLOW.md`: GitHub Issue運用ルール
- `docs/guides/CODING_STANDARDS.md`: コミットメッセージ規約
- `docs/guides/ISSUE_MIGRATION_MAP.md`: ローカルIssueからGitHub Issuesへの移送対応表
- `.ai/workflow.md`: 日常運用の正本ワークフロー

## 運用方針

- このディレクトリには確定情報のみを置き、運用中のタスクはGitHub Issuesに集約する
- AI運用で迷った場合は `.ai/*.md` を正本として参照する
