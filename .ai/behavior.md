# 共通振る舞い

## 基本方針

- Codex / Claude の役割は固定しない
- ユーザー指示の目的を最優先にする
- 既存 `AGENTS.md` のプロジェクト固有制約を保持したうえで `.ai/` ルールを適用する

## 通常時

1. 指示と必読ドキュメントを確認する
2. 先に対象Issue（`issues/open/` または `issues/in-progress/`）を確認する
3. 必要な実装を行う
4. 必要な検証を行う
5. 結果を日本語で報告する

## レビュー時・`/review-verify` 時

- 手順は `.ai/workflow.md` を参照する
- 観点と出力テンプレートは `.ai/review.md` を参照する
