# レビュー検証（/review-verify）

## 目的

`.context/_review_feedback.md` を基準に、レビュー指摘の妥当性を検証し、採用した指摘のみ修正・テストする。

## 実行手順

1. `.context/_review_feedback.md` の有無を確認する。
2. ファイルがない場合:
   - 修正作業は行わない。
   - 「レビュー指摘なし」として日本語で報告する。
   - `.context/_review_feedback.md` の出力有無は「出力なし」と明記する。
3. ファイルがある場合:
   - 各指摘を **採用 / 不採用 / 追加情報必要** で分類して妥当性を検証する。
   - 採用した指摘のみ修正を実施する。
   - 必要なテストを実行し、失敗したら修正して再実行する。
4. すべての修正・テストが完了したら `.context/_review_feedback.md` を削除する。
5. 最終報告は日本語で行い、以下を必ず含める。
   - 指摘ごとの判定（採用 / 不採用 / 追加情報必要）
   - 実施した修正内容
   - テスト結果
   - `.context/_review_feedback.md` の削除有無

## ルール

- レビュー内容は鵜呑みにせず、必ず自分で妥当性を検証する。
- 修正対象は採用した指摘のみとする。
- `/commit` または `/commit!` の明示的指示がない限りコミットしない。
